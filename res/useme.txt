How to use the application?
- Click on load image to load an image of any format (jpg, png, ppm) on the screen.
- Perform any operation , eg. Blur on the image.
- This is prompt a dialog to choose between apply to full image or split view.
- Split view option will open a slider to enter an integer spli percentage.
- A new window with the split is pop-up where one can toggle preview between image before and after the effect.
- User can either apply or discard the changes.
- The image can be saved using the save image button at any time.
- Loading a new image without saving the existing changes will prompt the user to save.

When invoked in this manner the program will open the graphical user interface. This happens if you simply double-click on the res file.
The GUI has image manipulation buttons on the right side of the frame and three buttons on the bottom:
- “Load Image” :
1. User opens the program clicks Load Image button
2. Select the required image from the files.
3. Select "Open Image" to load an image into the program.

-“Save Image”:
1. If the user clicks Save Image button, new window for save opens and the user needs to enter the File name with extension as well.
2.The user should also select the file type in the drop down option available such as ppm, jpeg, png.
3.The image gets saved in the location specified by the user with the given file name and extension.

-“Revert to Original”:
When the user clicks this button the image displayed reverts back to the original image loaded by the user.

Image Manipulation Buttons
1. Blur Operation.
-When u click on blur, the user gets the option to choose between split preview and apply effect to whole Image
-If the user clicks Split View: the user has access to the slider to select the split percentage
- Once the user selects the split percentage a new window pops up with the split image
- In this window there are three buttons in the bottom:
        —“Show original image” checkbox: The user can view the image with the split applied and has the option to toggle between the image with and without the effect using the checkbox
        —“Apply”: applies the effect to the whole image in the main window
	      —“Discard ”: reverts back to the image before the effect was applied
